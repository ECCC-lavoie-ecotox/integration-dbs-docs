Consolidation de la nomenclature des analytes

Auteur·rice

Steve Vissault

Date de publication

28 mars 2024

Nous désirons consolider la nomenclature des analytes. On peut observer plusieurs divergences dans la facon de nommer les analytes, exemple PCB-17 ou PCB 17. Cette section documente la démarche pour tendre vers une réconciliation des termes utilisés.

On charge les données intégrées depuis le package toxbox. Les données ne sont pas entreposées dans le package, la function toxbox::dbs_integration() permet de lire les fichiers Excels entreposées sur le disque réseau Z: et les fusionne dans un seul data.frame. Il faut pour cela que le VPN soit ouvert, si vous n’êtes pas sur site.

devtools::install_github("ECCC-lavoie-ecotox/toxbox")

    dplyr::mutate(original_name = variable) |>
    dplyr::mutate(
        variable = janitor::make_clean_names(variable, allow_dupes = TRUE, case = "none") |> tolower()

measurements_with_cas <-  analytes_cleaned |>
 dplyr::right_join(
    measurements_cleaned,
    by = c("Analyte" = "variable")
)
